# Tabla y Formulario con HTML y CSS

Proyecto realizado para una entrega de clase del Bootcamp de Eurofirms.

Consiste en una pagina web sencilla que muestra:

- Una tabla con un listado de usuarios.
- Un formulario de registro de usuario.
- Un diseno responsive basico con HTML5 y CSS3.

## Tecnologias utilizadas

- HTML5
- CSS3

## Estructura del proyecto

- `index.html`: estructura principal de la pagina.
- `style.css`: estilos de la interfaz.

## Objetivo de la practica

Practicar la creacion de tablas, formularios y estilos visuales basicos con una maquetacion limpia y adaptable.
