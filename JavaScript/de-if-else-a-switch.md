Pseudo código para pasar de cadena de if-else a switch:
=======================================================

if (color == RED) {
    ...
} else if (color == BLUE) {
    ...
} else if (color == GREEN) {
    ...
} ...

     ||
     \/

switch (color) {
    case RED: ....
    case BLUE: ....
    case GREEN: ...
        :
        :
}
